# My Wallet

Manual personal finance command center inspired by a premium blue leather wallet.

## v1 features

- First-time setup with first-name greeting
- Dashboard with net worth, free cash, reserved funds, and credit card debt
- Account-centric Banking page
- Sinking funds inside each bank account
- Credit cards with utilization and available credit
- Transactions for expenses, income, and balance adjustments
- Transfers between accounts, sinking funds, and credit card payments
- Goals linked to sinking funds
- Categories in Settings
- Local backup/export
- Installable PWA shell

## Design direction
